# NERDS — Listagem e Cadastro de Membros

Aplicação web em Vue 3 para cadastrar, listar e buscar membros do NERDS. Inclui:
- Formulário com validação de campos obrigatórios (Nome, Matrícula, E-mail).
- Lista/tabela reativa dos membros cadastrados.
- Campo de busca simples (nome, matrícula ou e-mail).
- Interface responsiva com CSS puro.

## Requisitos

- Node.js 18+ e npm ou pnpm

## Instalação

```bash
# Clonar o repositório
git clone https://seu-repositorio/nerds-membros.git
cd nerds-membros

# Instalar dependências
npm install
# ou
pnpm install
